# Shoe Store React App

This project is a simple React application where users can browse shoes and add them to a shopping cart.

## Features

- Display list of shoes
- Add items to cart
- Increase/decrease quantity
- Remove items
- Calculate total price using useState

## Technologies Used

- React
- JavaScript
- Vite

## How to Run

1. Install dependencies:
   npm install

2. Start server:
   npm run dev

3. Open in browser:
   http://localhost:5173
